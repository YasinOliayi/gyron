# `gyron`

`gyron` یک کتابخانه‌ی کاملاً **Async** برای ساخت ربات‌های پیام‌رسان بله با زبان Python است. این کتابخانه با هدف ساده‌تر کردن توسعه‌ی ربات‌ها و ارائه‌ی یک رابط ساده، مدرن و منعطف برای `Bot API` بله طراحی شده است.

این کتابخانه با بهره‌گیری از قابلیت‌های `async` و `await`، امکان توسعه‌ی ربات‌های سریع، مقیاس‌پذیر و قابل نگهداری را فراهم می‌کند و تلاش می‌کند تجربه‌ای روان و لذت‌بخش برای توسعه‌دهندگان ایجاد کند.

## ویژگی‌ها

- کاملاً مبتنی بر `async` و `await`
- پشتیبانی از `Long Polling` و `Webhook`
- Type Hint کامل
- منتشرشده تحت مجوز `MIT`

## نصب

برای نصب `gyron` دستور زیر را اجرا کنید:

```bash
pip install gyron
```

## شروع سریع


```python
from gyron.bot import BotClient
import asyncio


app = BotClient('TOKEN')


@app.on_update()
async def echo(update):

    await app.reply(update.text)

  
asyncio.run(app.run())
```
