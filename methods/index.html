BotClient

کلاس اصلی برای مدیریت ارتباط با API ربات بله. این کلاس وظیفه مدیریت ارتباط با API، ثبت هندلرها، مدیریت Middlewareها و دریافت آپدیت‌های ربات را بر عهده دارد.

پارامتر نوع توضیحات
token str اجباری - توکن ربات برای برقراری ارتباط با API بله.
use_webhook bool اختیاری - مشخص می‌کند ربات از Webhook استفاده کند یا Polling. مقدار پیش‌فرض False است.

on_update

یک دکوراتور برای ثبت هندلرهای دریافت آپدیت. هر تابعی که با این دکوراتور ثبت شود، هنگام دریافت آپدیت و در صورت عبور از فیلترهای مشخص‌شده اجرا خواهد شد.

پارامتر نوع توضیحات
*filter_func - اختیاری - یک یا چند فیلتر که قبل از اجرای هندلر بررسی می‌شوند.

middleware

یک دکوراتور برای ثبت Middlewareهای ربات. Middlewareها قبل از اجرای هندلر اجرا می‌شوند و می‌توانند عملیات موردنیاز را قبل و بعد از اجرای هندلر انجام دهند. برای ادامه روند پردازش، باید تابع call_next() فراخوانی شود.

پارامتر نوع توضیحات
- - این متد پارامتر ورودی ندارد.

نمونه استفاده

from gyron.bot import BotClient
import asyncio


app = BotClient('TOKEN')


@app.middleware()
async def middleware(update, call_next):

    print('before handler')
    await call_next()
    print('after handler')


@app.on_update()
async def echo(update):

    await app.reply(update.text)


asyncio.run(app.run())

get_updates

آپدیت‌های جدید ربات را از طریق متد getUpdates از API بله دریافت می‌کند.

پارامتر نوع توضیحات
offset int اختیاری - شناسه اولین آپدیتی که باید دریافت شود.
limit int اختیاری - حداکثر تعداد آپدیت‌هایی که در هر درخواست دریافت می‌شوند.
timeout int اختیاری - مدت زمان انتظار سرور برای Long Polling (بر حسب ثانیه).
request_timeout int اختیاری - حداکثر مدت زمان انتظار برای ارسال درخواست HTTP (بر حسب ثانیه).

خروجی: dict[str, Any]

دیکشنری شامل پاسخ API و لیست آپدیت‌های دریافت‌شده.

markup Union[InlineKeypad, Keypad] اختیاری - کیبورد اینلاین یا معمولی که همراه پیام ارسال می‌شود.
reply_to_message_id int اختیاری - شناسه پیامی که پیام جدید در پاسخ به آن ارسال می‌شود.

ربات را اجرا می‌کند. بسته به تنظیمات، ربات در حالت Polling یا Webhook شروع به دریافت و پردازش به‌روزرسانی‌ها خواهد کرد.

پارامتر نوع توضیحات
timeout int اختیاری - مدت زمان انتظار برای دریافت به‌روزرسانی‌ها در حالت Polling. مقدار پیش‌فرض 15 ثانیه است.
sleep float اختیاری - مدت زمان توقف بین درخواست‌های Polling زمانی که به‌روزرسانی جدیدی وجود ندارد. مقدار پیش‌فرض 0.7 ثانیه است.
webhook_path str اختیاری - مسیر Endpoint مربوط به Webhook. مقدار پیش‌فرض /webhook است.
host str اختیاری - آدرس میزبان سرور Webhook. مقدار پیش‌فرض 0.0.0.0 است.
port int اختیاری - پورتی که سرور Webhook روی آن اجرا می‌شود. مقدار پیش‌فرض 8000 است.

نحوه استفاده از وبهوک

from gyron.bot import BotClient
import asyncio


token = 'TOKEN'
webhook_url = 'https://dgjzs-217-218-35-75.free.pinggy.net/webhk'


app = BotClient(token, use_webhook=True)


@app.on_update()
async def start(update):

    await app.reply('سلام')

async def main():

    res = await app.set_webhook(webhook_url)

    if res:
        print('started')
        await app.run(webhook_path='/webhk')


asyncio.run(main())
